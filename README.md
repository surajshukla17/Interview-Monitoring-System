# Interview Monitoring System Using Image and Audio Processing

This project monitors online interviews using real-time image and audio processing.

## Features
- Multiple Face Detection
- Multiple Voice Detection
- Face Movement Detection
- Real-time Monitoring
- Final Summary Report

## Tools & Technologies
- Python
- OpenCV
- NumPy
- SoundDevice

## How to Run
pip install -r requirements.txt  
python main.py
